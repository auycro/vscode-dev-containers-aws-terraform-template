## Example usage

Open this folder in the container.

Check the Version

```sh
$ terraform -version
$ aws --version
$ aws-iam-authenticator version
$ kubectl version
$ helm version
$ minikube version
```

Setting up aws cli

```sh
$ aws configure
```
